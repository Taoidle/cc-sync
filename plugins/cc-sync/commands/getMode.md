# 查看同步模式 (Get Sync Mode)

使用 MCP Server 的 `getMode` 工具查看当前同步模式和将要同步的项目列表。

## 使用示例 (Examples)

```
# 查看当前同步模式
/cc-sync:getMode
```

---

Use the `getMode` tool from the cc-sync MCP Server to view the current sync mode and list of items to sync.

## Examples

```
# View current sync mode
/cc-sync:getMode
```
