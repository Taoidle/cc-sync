# 查看同步模式 (Get Sync Mode)

使用 cc-sync MCP 服务器的 `getMode` 工具查看当前同步模式和将要同步的项目列表。

## 执行步骤

1. 调用 cc-sync MCP 服务器的 `getMode` 工具（不需要参数）
2. 显示返回的模式和同步项目列表

## 示例

```
/cc-sync:getMode
```
